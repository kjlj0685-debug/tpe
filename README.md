# Refactoring Lab

This project demonstrates Java refactoring examples with Gradle and JUnit.

## Project structure

- `src/main/java/org/example` – example source code
- `src/test/java/org/example` – unit tests

## How to run

If you have Gradle installed:

```bash
gradle test
```

If you use the Gradle wrapper, run from the project root instead:

```bash
./gradlew test
```
